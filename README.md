# CTF Writeups

A personal collection of Capture The Flag (CTF) challenge writeups and solutions, documenting my methodology, tools and thought process across multiple platforms.

---

## Platforms Covered

| Platform | Focus Area | Difficulty |
|---|---|---|
| [TryHackMe](https://tryhackme.com) | Guided learning paths, rooms | Beginner to Intermediate |
| [HackTheBox](https://hackthebox.com) | Real-world machine exploitation | Intermediate to Advanced |
| [PicoCTF](https://picoctf.org) | Academic CTF puzzles (Carnegie Mellon) | Beginner to Intermediate |
| Other / Mix | Various open CTF competitions | Mixed |
